Recyclable Waste Classifier

Recyclable waste classification model with voting mechanisms, trained on multimodal sensor data

As a data scientist intern, I contributed to a project focused on developing a deep neural network model for classifying recyclable waste into four categories: cans, clear PET bottles, opaque PET bottles, and glass bottles, given the multi-sensory data including pictures from 5 varying angles, weights, metal content, and object opacity. Inspired by research that employs voting mechanisms for multiclass classification (Zhang et al., 2020) (Leon et al., 2017), a similar voting-based ensemble was used for multiclass classification on multimodal sensor data.

Simplified Architecture of the multimodal deep neural network model for recyclable waste classification

My primary responsibilities covered two key areas:

  • 1) data pipeline construction, which included data collection, creating a data collection manual for crowdsourcing, and data preprocessing.

  • 2) model development and evaluation, which involved model design, training, and testing. The implementation was done using the PyTorch framework, AWS ECS/EC2, and Docker.
